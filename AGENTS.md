## Scientific gates

- Tests are evidence about code behavior, not truth.
- A scorer is not role-free if it can access `truth_role`, expected outcomes,
  designed labels, semantic candidate identity, or legacy role-derived outputs.
- Summary booleans such as `manifest_frozen=true` are not proof. Verify source
  artifacts and hashes or report `not_verified`.
- Zero false crowns cannot pass by itself. The evaluator must be capable of
  exposing an injected false crown, and always-HOLD must fail.
- Nested triad27/deep81/wide243 cases must not be pooled as independent evidence.
  Report unique cases, denominators, rates, overlap, and uncertainty.
- Prior-touch support is implemented as a development scorer, but it has no
  scientific authority until locked evaluation shows claim-relevant behavior
  and the final frozen verdict consumes it. Do not rename it continuous
  persistence.
- A failed frozen holdout is preserved and retired, not converted into a tuning
  set.
