# License and Citation Finalization

**Status:** v1.0.2-alpha public-repo readiness patch.

## Expand — public reuse door

ZeroGateSim now uses the MIT License for the source repository.

This makes the software reusable, forkable, inspectable, and testable by others without leaving the public repo behind a fog-door.

## Witness — citation metadata

`CITATION.cff` now points to the intended public repository:

`https://github.com/S1MB4-CLFS/zerogate_sim`

The citation metadata intentionally does not include a DOI yet. The DOI should be added only after a Zenodo archive exists.

## Resist — do not overclaim through license

The MIT license is a software reuse license. It does not make the theory proven, and it does not license future paper deposits automatically if those records specify a different license.

Future manuscript/PDF deposits may use their own explicit license, likely CC-BY-4.0 if the goal is public scholarly reuse.

## Current boundary

The public repo may say:

> ZeroGateSim v1.0-alpha passed a first-research-alpha toy-field proof-of-concept, demoting false-one pressure with zero final false-one crowns across original and fresh-seed trinary adversarial proof records.

The public repo must not say:

> ZeroGateSim proves cosmology, physics, or final trinary mathematics.
