# ZeroGateSim Theory Kernel

**Version:** `v0.2.1-alpha`  
**Status:** speculative, test-shaped, not proven physics

## 1. Core sentence

A dimension is stabilized freedom that has passed through zero without losing coherence.

## 2. Why this exists

The usual dimensional ladder describes completed geometry:

- point,
- line,
- plane,
- volume,
- time.

That is useful for teaching coordinate structure. It is incomplete as a genesis model.

The zero-gate frame asks a prior question:

> What allows any dimension to become expressible, measurable, stable, and countable?

## 3. Time as generative ordering

Time is not treated here merely as a later coordinate.

Time is the generative ordering condition through which dimensions become expressed.

This does not deny standard spacetime physics. It separates coordinate description from emergence language.

## 4. Four gates

### Distinction

Something becomes separable from background.

Without distinction, there is no candidate.

### Polarity

The distinction gains positive and negative expression around zero.

Without polarity, there is push but no axis.

### Relation

Polarity becomes bound into structure with other candidate freedoms.

Without relation, polarity can split, drift, or oscillate without becoming part of a field.

### Return

The expressed structure folds back toward zero while preserving coherence.

Without return, expansion is not proven stable. It may only be escape, noise, or temporary force.

## 5. Return-potential

Return is not arbitrary.

The first three gates generate return-potential.

Distinction separates.

Polarity tensions.

Relation binds.

When binding becomes coherent, expansion curves back as return.

Formal placeholder:

```math
\Gamma_i(t)=g_D^i(t)g_P^i(t)g_R^i(t)
```

`Gamma` is not physical gravity. It is a gravity-like coherence prompt.

It asks whether distinction, polarity, and relation together predict return.

## 6. Zero-gate coherence

The gate coherence of candidate freedom `i` is:

```math
C_Z^i(t)=\min(g_D^i(t),g_P^i(t),g_R^i(t),g_B^i(t))
```

The minimum rule protects against fog.

A missing gate cannot be averaged away.

## 7. Expressed dimension count

A candidate freedom counts as expressed only if it has measurable strength and enough zero-gate coherence:

```math
d_Z(t)=\sum_i H(\sigma_i(t)-\epsilon)H(C_Z^i(t)-\theta_Z)
```

where `H` is a threshold function.

## 8. Zero-depth

Zero is not one flat absence.

Zero may have depth.

A candidate does not deserve to be counted as one merely because it appeared.

It earns expression through coherent returns.

Depth states:

- `Z^(0)` raw zero,
- `Z^(1)` zero A: distinction survived return,
- `Z^(2)` zero B: polarity survived return,
- `Z^(3)` zero C: relation survived return,
- `Z^(4)` zero D: return itself survived return.

A real one is not the first thing after zero.

A real one is what zero can return as without lying.

## 9. Simulation hypothesis

Candidate freedoms with high `g_D * g_P * g_R` should be more likely to show coherent return.

Candidate freedoms with high `C_Z` should be more likely to become stable expressed freedoms than candidates selected by amplitude, polarity, relation, return, average gate, or random pressure alone.
