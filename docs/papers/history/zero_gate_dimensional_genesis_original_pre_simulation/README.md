## Why preserve it?

The original manuscript records the conceptual origin of the work: zero-zone, generative ordering, distinction, polarity, relation, return, return-depth, and the first simulation target.

The software later tested and repaired parts of that frame. Preserving the original manuscript makes the lineage honest:

```text
original seeing -> executable simulation -> proof-of-concept record -> simulation-supported paper
```
