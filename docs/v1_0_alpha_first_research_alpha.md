# ZeroGateSim v1.0-alpha — First Research Alpha

**Status:** first research-alpha release freeze  
**Core mutation:** none to the four-gate law  
**Release posture:** proof-of-concept passed inside generated toy fields

v1.0-alpha freezes the first proof-shaped state of ZeroGateSim.

It does not claim cosmology, physics, or final trinary mathematics. It claims a narrower and stronger thing: the current simulation grammar can separate earned-one from raw expression, latent overcrown, and false-one pressure under trinary adversarial pressure, and the result reproduced on fresh seeds.

A real one is not what appears first. A real one is what zero can return as without lying.
