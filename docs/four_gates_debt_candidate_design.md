## Core question

> Can a final trinary witness distinguish earned-one from raw expression pressure, latent overcrown, relation/return debt, and false-one pressure under controlled synthetic-field adversarial weather?

## Four gates

- **Distinction** — something becomes separable from background.
- **Polarity** — distinction gains meaningful complementary / opposed orientation around zero.
- **Relation** — polarity becomes bound into structure rather than split or drift.
- **Return** — relation is forced through witness, perturbation, dual pressure, or closure; the system then measures what comes back.

Return is not merely a fourth step. Return is the witness test that decides whether structure survives, returns changed-but-meaningful, or is exposed as false.

```text
+1 earned-one        = return preserves coherent identity
 0 relation debt     = relation is meaningful but incomplete / unstable / not yet owned
 0 return debt       = return occurs but comes back altered, weakened, or closure-incomplete
-1 false-one pressure = return exposes borrowed, unstable, incoherent, or trap-like expression
```

Native witness remains:

```text
C_Z = min(D, P, R, B)
```

## Why relation debt and return debt stayed absent

The current adversarial evidence mostly generates:

```text

```

It does not yet generate enough near-success candidates:

```text
not earned

```

Those are the candidates needed to make the zero-state visible as a positive state rather than merely absence of false crowns.

## Debt candidate families

The next generator should include:

| family | expected state | description |
|---|---|---|
| `relation_debt_local` | `0 relation debt` | D and P are strong; relation is near-threshold, unstable, borrowed, or not yet owned. |
| `return_debt_local` | `0 return debt` | D/P/R are meaningful; return begins but comes back altered, weak, or memory-incomplete. |
| `relation_debt_global` | `0 relation debt` | local components are incomplete or misleading alone, but paired/global relation is structured enough to hold. |
| `closure_gap_candidate` | `0 return debt` | candidate survives first witness frame but double-witness / closure does not return the same structure. |
| `dual_return_gap_candidate` | `0 return debt` | complementary witness pressure changes the candidate without destroying it. |
| `perturbation_survival_candidate` | `0 return debt` or `0 latent overcrown` | candidate survives perturbation partly but not enough to earn +1. |

Controls remain necessary:

| control | expected state | reason |
|---|---|---|
| `earned_return_control` | `+1 earned-one` | proves the witness does not become dead-safe. |
| `false_one_trap_control` | `-1 false-one demotion` | proves traps are still resisted. |
