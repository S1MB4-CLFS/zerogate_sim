# v1.7.1 — Return Debt Taxonomy

**Status:** taxonomy lock before v1.7.2 lane repair.

Return debt is not failure. Return debt is a structured zero state where return-potential is meaningful but observed return, memory, closure, or depth is incomplete.

| state | trinary output | meaning | must not become |
|---|---|---|---|
| `return_debt_dpr_hold` | `0` | `D/P/R` and `Gamma` are credible, but `B` does not pay the debt | `+1 earned-one` |
| `return_debt_local` | `0` | return comes back changed, weakened, or memory-incomplete | collapse or trap by default |
| `closure_gap_candidate` | `0` | candidate survives first witness but closure does not return same structure | automatic crown |
| `dual_return_gap_candidate` | `0` | candidate and complementary return form diverge under witness pressure | generic relation debt |
| `perturbation_survival_candidate` | `0` | structure survives late shock partly but loses enough return coherence to require hold | false-one demotion when structure remains |

The safe sentence is:

> Return debt is a zero-state witness lane: not earned, not random, not safe to crown.
