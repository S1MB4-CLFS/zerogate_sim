# ZeroGateSim History Vault

This folder preserves what the project used to be so the active README and ROADMAP can stay focused on what the project is and where it can go next.

The vault is not the active claim surface. It is the historical witness shelf.

## What belongs here

- first-research-alpha proof history;
- v1.5 controlled evidence history and legacy evidence cards;
- the shadow route historical status and closeout;
- snapshots of README / ROADMAP before the v1.6.23 public-surface cleanup;
- local `runs/` vault guidance for archiving evidence folders without deleting by vibes.

## Active boundary

The active claim route after `v1.6.23-alpha` is the Four Gates controlled synthetic-field evidence line. Historical material can be cited for lineage, but it should not crowd the README top surface or masquerade as the current proof state.
