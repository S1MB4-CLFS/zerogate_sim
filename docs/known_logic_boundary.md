# ZeroGateSim Known-Logic Boundary

**Line:** `v1.3-alpha`
**Status:** projection mirrors only

ZeroGateSim does not claim identity with any known logic family.

It is not Kleene logic, Lukasiewicz logic, Priest logic, Belnap logic, fuzzy logic, balanced ternary arithmetic, or a replacement for those systems.

The purpose of the known-logic line is narrower:

> Project native ZeroGateSim states and scores into nearby formal mirrors to see what is preserved, what is collapsed, and what is distorted.

## Native layer comes first

ZeroGateSim's native stack is:

```text
candidate pressure signal
-> distinction / polarity / relation / observed return
-> C_Z = min(D, P, R, B)
-> raw expression pressure
-> return-depth / temporal lineage / echo-independence / role witness
-> final trinary output
```

Known logics are used as mirrors after that native stack is defined. They do not authorize a crown.

## Projection versus identity

A projection is a translation for comparison.

An identity claim says the systems are the same.

Only projection claims are allowed in this line.

Allowed:

> The fuzzy mirror compares ZeroGateSim's weakest-gate coherence with continuous conjunction variants such as min, product, average, and Lukasiewicz-style conjunction.

Forbidden:

> ZeroGateSim is fuzzy logic.

Allowed:

> A Belnap-style mirror may help report evidence-for, evidence-against, both, and neither.

Forbidden:

> ZeroGateSim proves Belnap logic or is equivalent to Belnap logic.

## Mirror order

The mirror order is deliberate:

1. **Fuzzy / many-valued scoring mirror** — closest to current continuous gate scores.
2. **Belnap evidence-state mirror** — closest to raw expression versus false-one / hold evidence.
3. **Paraconsistent conflict-locality mirror** — tests whether conflicting pressure stays local.
4. **Kleene / Lukasiewicz compression mirror** — tests what is lost when ZeroGateSim collapses final output to true / unknown / false.


Allowed:

> A three-valued mirror may show what is lost when final `+1 / 0 / -1` output is compressed to `true / unknown / false`.

Forbidden:

> ZeroGateSim is Kleene K3 or Lukasiewicz L3.

## Required loss report

Every mirror must state what it cannot preserve.

Examples:

- The fuzzy mirror can grade continuous pressure but cannot crown earned-one without return-depth, temporal lineage, echo-independence, and role witness.
- A three-valued mirror collapses `0+`, `0`, and `0-` into one middle state.
- The paraconsistent mirror can report conflict locality but does not by itself define dimensional genesis, decide final truth, or license contradiction as a crown.

## Operating sentence

Known logic mirrors are pressure tests, not borrowed authority.

If a mirror makes ZeroGateSim look stronger by hiding a native wound, the mirror failed its witness job.
