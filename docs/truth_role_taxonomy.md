# Truth-Role Taxonomy and Echo-Mimic Diagnostic

**Version introduced:** v0.2.13-alpha

## Purpose

The early corpus used a binary label: designed stable / not designed stable. The wide243 + triad27 run showed that this was too crude.

Some candidates are not supposed to express immediately. They are latent probes. Calling them false negatives is bad witness.

Some candidates should never express. If they do, that is breach pressure.

This version repairs the candidate truth layer without mutating the core gate.

## Roles

### +1 expresser

A candidate intended to earn expression under suitable weather.

### 0 latent/probe

A candidate intended to hold, mature, wait, or expose boundary pressure. It is not failure if it remains in zero.

### -1 trap

A candidate intended to be rejected or quarantined. If it becomes +1, that is a breach.

## Echo-mimic diagnostic

Echo-mimic score asks whether a candidate is strongly explained by the leave-one-out field average.

This is a diagnostic, not a gate. It marks candidates that may be borrowing field coherence rather than earning independent return.

Echo becomes dangerous when a trap is crowned as expression.
