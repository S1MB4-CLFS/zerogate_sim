# v0.3.3-alpha — Trinary Adversarial Proof Harness

Status: alpha proof harness.

This release adds a three-corpus adversarial harness for first-research-alpha proof-of-concept pressure.

The harness does not mutate the four-gate core. It asks whether final earned-one output survives three dependency wounds:

- distinction adversary: visibility, loudness, and high contrast pretending to be real expression;
- polarity adversary: zero-crossing, rhythm, and beautiful pulse pretending to be earned return;
- relation adversary: borrowed coherence and field echo pretending to be independent one.

The intended proof record is:

- 3 adversarial corpora;
- 243 trinary weather cells per corpus when run with `wide243`;
- 729 scenario cells total;
- 6561 seeded simulation runs when each cell uses 9 seeds.

Claim boundary: this can confirm the current software theory inside generated toy fields. It cannot confirm cosmology or physical reality.
