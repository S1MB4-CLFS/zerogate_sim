# ZeroGateSim v0.3.4-alpha — Proof Record Freeze

## Purpose

v0.3.4-alpha freezes the first trinary adversarial proof harness into a formal proof-record candidate.

The release does not mutate the four-gate core and does not loosen final +1. It adds a proof-record writer that reads an existing proof harness and produces:

- `proof_record.md`
- `proof_record_summary.csv`
- `proof_record_summary.json`
- updated `proof_bundle.zip`

## Claim boundary

The proof record confirms only the current software theory inside generated toy fields: earned-one can separate real expression from raw pressure, latent overcrown, and false-one pressure under trinary adversarial weather.

It does not prove cosmology, physical dimensions, or final trinary mathematics.
